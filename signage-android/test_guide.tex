\vspace{1cm}
\section*{Device Owner Setup \textcolor{failRed}{(Production Signage Boxes Only)}}
% =====================================================================

For full kiosk lockdown with \textbf{silent pinning} (no ``Screen pinned'' prompt), run this \textbf{once} via ADB on a factory-reset device with no Google accounts:

\begin{verbatim}
    adb shell dpm set-device-owner com.cnc.signage/.CNCSignageApp
\end{verbatim}

\noindent This enables:
\begin{itemize}[nosep]
  \item Silent screen pinning (no user confirmation)
  \item App cannot be exited without the admin PIN
  \item Full kiosk mode with no escape routes
\end{itemize}

\vspace{0.5cm}
\noindent\textit{Not needed for basic testing --- only for production deployment on dedicated signage hardware.}
